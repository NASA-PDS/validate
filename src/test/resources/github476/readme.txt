Bundle Description for the M2020 SPICE Archive Bundle
=====================================================

Overview
========

   The Mars 2020 SPICE archive bundle contains observation geometry and
   other ancillary data in the form of SPICE System kernel files for
   the Mars 2020 rover, its instruments, and targets.

   The detailed description of the M2020 SPICE archive is provided
   in the latest version of spiceds_v*.html included in the
   document collection.


Document Collection
===================

   The document collection contains documents describing the M2020
   SPICE archive, PDS4 collection labels, and PDS4 collection
   inventory tables.


SPICE Kernels Collection
========================

   The SPICE Kernels collection contains M2020 SPICE kernels and
   meta-kernels, PDS4 collection labels, and PDS4 collection
   inventory tables.


References
